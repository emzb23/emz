Klippa och klistra

En sak som är väldigt bra för att snabba upp datoranvändande är snabbkommandon för att klippa och klistra. När du klipper ut något hamnar det i cyberrymden där det stannar tills du klipper ut något nytt.

  • CTRL + C Kopiera
  • CTRL + X Klipp ut
  • CTRL + V Klistra in
  • CTRL + S Spara
  • CTRL + O Öppna
  • CTRL + F Sök
  • CTRL + A Markera allt
  • CTRL + SHIFT Lägg till markering
  • WINDOWS + P Växla mellan skärmar
  • WINDOWS + L Lås datorn
  • ALT + TAB Växla mellan fönster
  • F5 Ladda om hemsida (inom webbläsare)

Markera en eller fler filer

Om man klickar på en fil får den en mörk bakgrund. Det betyder att den är markerad.

Genom att hålla inne vänster musknapp och dra med musen kan man markera flera filer.

Vill du markera flera filer som är sammanhängande kan du använda dig utav SKIFT tillsammans med musklick.

Vill du markera flera filer som inte är sammanhängande kan du använda dig utav CTRL + musklick.
